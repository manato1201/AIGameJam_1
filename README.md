# Welcome to AIGameJam !

## 大学で開催されたバイブコーディングを用いてゲームを作成するという企画で作ったものです

## 3DFishingゲーム

### 製作時間：2時間

レベルで新規マップが解放される機能、図鑑機能、ショップ機能、サバイバル機能といった多くの機能を詰め込んでいます

自身の考えたゲームシステムが面白いか確認するのに活用できるという知見を得られました

バイブコーディングをした際にエラーが起きましたが、エラーの原因が変数の未定義・再定義によるもの、スコープの範囲外といった問題が多かったです

これはJavaScriptによる特性の影響だったので、間違いが起こる傾向が見れてとても興味深かったです
